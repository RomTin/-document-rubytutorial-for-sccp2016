## 6 クラス

これまで何度か触れているように、Rubyはオブジェクト指向言語である。
この章ではオブジェクトとその枠組みであるクラスについて説明していく。
オブジェクト指向は、人・言語などによってまったく考え方が異なる。この資料では筆者のRubyにおけるオブジェクト指向の考え方を説明していくが、限られた考え方・機能しか説明しないため、必ず別途資料を参照して理解を深めて頂きたい。
